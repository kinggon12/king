简介

本项目 King 是一个高性能、易扩展的开源解决方案，适用于多种应用场景。

  • 高性能架构
  • 简单易用的接口
  • 社区驱动的开发

安装

你可以通过以下命令快速安装：

git clone https://github.com/kinggon11/king.git
cd king
npm install

使用方法

以下是一个简单的使用示例：

import { King } from 'king';

const king = new King();
king.run();
